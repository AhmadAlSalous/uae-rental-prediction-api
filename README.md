## UAE Rental Prediction API

A FastAPI + ML project that predicts rental prices across major UAE cities based on features like city, type, area, and furnishing.

## Features
- FastAPI backend serving ML model
- Interactive frontend with dynamic dropdowns
- Model trained on UAE rental dataset

## How to run
1. Install dependencies: `pip install -r requirements.txt`
2. Run API: `uvicorn app.main:app --reload`
3. Visit: [http://127.0.0.1:8000](http://127.0.0.1:8000)
